# Perubahan — Update Keempat

## 🔧 Struktur file diratakan (tanpa folder)
Ternyata uploader GitHub yang kamu pakai hanya menerima file individual, tidak bisa folder. Jadi semua file ikon/screenshot yang tadinya di dalam folder `icons/` sekarang dipindah ke root bersama file lain — semuanya file lepas, tidak ada folder sama sekali. Tinggal select semua file lalu upload sekaligus.

Folder `source/` (kode mentah .jsx untuk pengembangan) dihilangkan dari paket wajib-upload — itu opsional, hanya untuk disimpan sendiri kalau butuh, tidak dipakai GitHub Pages.
